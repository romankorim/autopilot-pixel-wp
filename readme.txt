Auto-installed by Autopilot. Fires Meta Pixel on page views and forwards WooCommerce purchases to the Meta Conversions API server-side.

== Description ==

This plugin is installed automatically by Autopilot (https://app.iteractive.ai) when a customer connects their WordPress site. It does two things:

1. Fires the Meta Pixel on every page load.
2. Forwards completed WooCommerce orders to Meta's Conversions API server-side, so iOS-14 attribution stays intact.

The Pixel ID is configured automatically by Autopilot during setup — you do not need to touch any settings.

== Removing the plugin ==

Deactivate and delete from your WordPress admin. Your site is otherwise untouched.
